# v3.277 可执行院校专业清单

## 目标

把方向级推荐结果收敛成家长可以逐项核验的院校专业清单，而不是把泛泛的学校举例误当作可填志愿。

## 规则

- 只汇总已命中的本省、同科类、结构化院校专业记录；没有数据记录的泛候选不进入清单。
- 同一院校专业在多个方向池命中时去重，并保留“命中多个方向”的信息。
- 专业录取边界按已校准的可达性分为优先核验、稳妥候选、冲刺候选。
- 招生计划和历史征集快照单列为计划与资格核验，始终不展示为录取概率。
- 每条清单继续保留年份、最低分、位次、数据时效和来源限制提示。

## 验证

`node scripts/test-application-plan-v3277.mjs` 验证同一院校专业跨方向去重、历史数据进入冲刺层、计划记录隔离和无结构化记录的泛候选排除。
