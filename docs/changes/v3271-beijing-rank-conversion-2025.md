## 目标与边界

接入北京教育考试院 2025-06-25 发布的《北京市2025年高考考生分数分布》10 页官方 PDF，把当前第三方图片待解析状态替换为可计算的官方位次换算。

本轮只增加 `rankConversions`，不新增院校或专业录取记录，不把一分一段累计人数解释为院校录取概率。698 分以上、379 分以下按官方合并分数段保留，不拆造逐分位次。

## 验收条件

- AC1：页面和 PDF 均来自 `bjeea.cn`，标题、发布日期、10 页页数与 SHA-256 固定。
- AC2：解析 347 行，累计人数逐行满足 `上一累计 + 本段人数 = 当前累计`，最终闭合到 65434。
- AC3：北京 2025 综合类位次换算从 0 增至 347；全国位次换算从 116309 增至 116656，录取/计划记录仍为 841776。
- AC4：第三方北京 2025 图片队列标记为被官方源替代，rank source 统计从 parsed/queued 136/67 更新为 137/66。
- AC5：31 省分片、mac_2T 镜像、4177 HTTP 与浏览器推荐回归全部通过。

## 审查修复

- 合并分段统一显示“同区间人数”，370/379、698/700、100/109 两端均通过回归；650 分逐分行仍显示“同分 102 人”。
- 全量构建器同时读取 `records` 和独立 `rankConversions`，347 条均保存 `dataType=rank-conversion` 与官方 `sourceQuality`。
- 高分本科画像排除专科记录；北京 650 分测试专科命中为 0，200 分专科画像仍保留 586 条记录；职业技术大学本科批 10 条保留、专科批 11 条隔离。
- “不接受高学费中外合作”会排除学费未知的合作记录；已知低学费合作项目不误伤。
- 独立 Reviewer 前两次提出的 P1/P2 均已关闭，第三次结论 PASS。
