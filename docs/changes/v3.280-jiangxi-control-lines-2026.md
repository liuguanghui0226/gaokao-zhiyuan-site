# v3.280 江西 2026 控制线

## 需求与边界

- 把江西 2026 普通类、特殊类型、三校生、艺术和体育控制线接入线上省份分片。
- 普通历史/物理本科线只用于本专科资格路由；控制线不得当作院校投档线、专业录取分或录取概率。
- 特殊类型、三校生、艺术和体育记录必须与普通考生路径隔离。
- 不绕过江西省综合管理平台登录，不抓取考生信息或受限的 2024-2025 专业组投档数据。

## 来源与实现

- 公开来源：江西省教育厅发布、阳光高考转载的《江西省2026年普通高校招生各类各批次录取控制分数线揭晓》。
- 导入器校验标题、发布日期、发布来源和九组正文断言，生成 30 条 `control-line` 记录及页面/正文 SHA256。
- 定向合并器只接受 v3.275 的 846432 条基线，原子更新江西 gzip 分片、核心索引和发布清单；重复来源或计数漂移会拒绝执行。
- 回滚边界是本次提交前的三份 gzip 文件；完整重建器不参与本次定向发布。

## 验证计划

- 校验 30 条记录分类为普通 4、特殊 2、三校生 2、艺术 20、体育 2。
- 校验普通历史本科/高职 479/220，普通物理本科/高职 412/200。
- 校验历史 478/479 与物理 411/412 的本专科路由边界。
- 运行当前全国回归，验证 31 省、155 个代表分段和所有 gzip 哈希。
- 部署后从 GitHub Pages 公网重新下载 31 省分片，并复核江西来源记录与两条普通本科线。
