# v3.309 齐鲁工业大学 2021-2025 全国录取数据

## 目标

从齐鲁工业大学本科招生网官方历年分数接口导入 2021-2025 年分省、分科类、分招生类型、分专业数据，为全国候选池补充学校直接公布的最低分、平均分、最高分、最低录取位次、控制线和包含专业。

## 官方来源

- 历年分数查询：<https://zsb.qlu.edu.cn/score>
- 官方字典接口：<https://zsb.qlu.edu.cn/home/api/biz_dict>
- 官方分数接口：<https://zsb.qlu.edu.cn/home/api/score>
- 依照省份、年份、科类和招生类型逐级查询，保存 718 份原始响应，共覆盖 297 个分数查询和 2157 条源行。
- 每份响应保存请求字段、字节数和 SHA-256；原始 HTML/JSON 与清单进入版本证据包，不进入 GitHub Pages 运行分片。

## 事实边界

- 新增 2157 条记录，覆盖 2021-2025 年和 28 个官网有实际记录的省级口径。
- 官网列有西藏、青海、宁夏入口，但五年内没有分数记录；这三个省明确保持来源空缺，不补空白行或推测数据。
- 1849 条普通记录为 `school-official-only`；308 条艺体、中外合作、地方专项、民族班、校企合作、菏泽校区等限定路径为 `special-path-only`。
- 2074 条最低录取位次来自学校官网接口，标记为 `rankEvidenceScope=school-recorded-min-score-rank` 和 `rankDerivedFromScore=false`；83 条源表无位次的记录保持 `rankUnavailable=true`。
- 官网分数接口没有选科要求和录取人数，运行层不补造；每条普通记录强制提示回当年招生计划和章程复核选科条件。
- 新疆 9 条记录全部是民族班且无最低位次，只进入特殊路径，不进入普通推荐。
- 学校官网单校录取边界不能替代省考试院全量投档/录取表，也不能单独生成录取概率。
- 公开站验收发现院校专业卡片尚未把考生城市偏好计入排序；本轮补入城市命中权重，并让“学校官网单校边界”和“官网未列选科要求”同时展示，避免数据已入库却被更安全但不合偏好的候选挤出前排。

## 验收

1. 718 份官方响应、297 个分数查询、2157 条源行和五年度计数完整，重复 ID 和跳过行均为 0。
2. 普通与八类招生路径严格隔离，原生最低录取位次与分数换算位次严格区分。
3. 31 省 gzip 分片、核心索引和清单哈希一致，运行总量为 859382 条。
4. 江西 2025 年网络空间安全样本保留最低分 529、平均分 538.35、最高分 551、控制线 429 和官网最低录取位次 52796；选科要求保持未公布状态。
5. 当前发布 48 项回归全部通过。
