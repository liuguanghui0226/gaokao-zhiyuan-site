# v3.320 广西 2025 双加分口径位次对齐

## 变更

- 导入广西 2025 普通类历史、物理一分一档表，并区分区外院校“全国性加分”和区内院校“全国性/地方性加分取最高”两种口径。
- 四张表形成 1896 条位次区间；1892 个公开分数档、7568 个单元跨图片、XLSX 和 HTML 核对，差异为 0。
- 使用教育部 2025 全国普通高校名单判断院校是否位于广西，覆盖全国 2919 所、广西 89 所院校。
- 为 8222 条广西 2025 普通整数最低分记录补充对应口径的省级位次，其中区外院校 7018 条、区内院校 1204 条。
- 广西考生表单可分别输入区外、区内投档分和位次；推荐器按目标院校所在地选择对应输入和位次表。

## 边界

- 110 条 `special-path-only` 记录不连接普通类位次。
- 最高分未逐分披露的四组记录只保存官方可确定的合并位次区间。
- 表中人数、累计人数和名次不包含已确定录取的考生。
- “最低分换算位次”是同年省级分数段区间，不冒充院校投档表直接公布的原生最低位次，也不提升原最低分来源等级。
- 正式填报仍须核验当年招生计划、选科要求、院校章程和广西招生考试院最新公告。
