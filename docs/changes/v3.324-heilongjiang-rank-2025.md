# v3.324 黑龙江 2025 普通类位次对齐

## 结果

- 新增黑龙江 2025 历史类 528 条、物理类 563 条文化课位次区间。
- 1091 个正人数分数档与考试院 XLS、黑龙江省政府发布页和两份完整表交叉核验。
- 1091 行、3273 个分数/人数/累计人数单元差异为 0。
- 为 7098 条普通整数最低分补充同年同科类省级位次，其中 5706 条最低分来自官方来源。
- 84 条特殊路径继续隔离；129 分及以下不外推。
- 全国运行层保持 868426 条结构化记录，位次换算增至 128036 条，来源说明增至 5128 条。

## 口径

官方明确说明一分段表不含照顾政策分。所有本轮关联记录均保存：

- `rankScoreBasis=gaokao-cultural-score-excluding-policy-bonus`
- `rankPolicyBonusIncluded=false`
- `rankEvidenceScope=score-derived-provincial-segment`
- `rankDerivedFromScore=true`

换算结果是同年省级文化课分数段区间，不是院校投档或录取原表直接公布的原生最低位次。

## 边界

- 历史类 659-750 分只保存 1-19 名合并档。
- 物理类 694-750 分只保存 1-34 名合并档。
- 历史类 654、144 分和物理类 136、134 分为零人数空档，不生成位次记录。
- 两科只公开到 130 分，129 分及以下不推算。
- 艺体综合分、科类不明、非整数分和特殊路径不连接普通文化课位次。
