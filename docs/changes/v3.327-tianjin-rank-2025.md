# v3.327 天津 2025 官方位次表与最低分换算

## 结果

- 接入天津 2025 普通高考总成绩 381 个分数档。
- 官方 PDF 与完整 HTML 比较 1143 个数值格，差异为 0。
- 官方标题和招生政策共同确认分数口径含政策加分。
- 3195 条天津 2025 综合普通类整数最低分获得省级位次区间，其中 2910 条来自官方来源。
- 20 条特殊路径、物理/历史标签、非整数分和 299 分以下记录不自动套表。
- 680 分及以上只保存公开的 1-656 名合并区间。
- 全国运行层保持 868426 条结构化记录，位次换算增至 128972 条，来源说明增至 5131 条。

## 口径

换算记录保存：

- `rankDerivedFromScore=true`
- `rankEvidenceScope=score-derived-provincial-segment`
- `rankScoreBasis=gaokao-total-including-policy-bonus`
- `rankPolicyBonusIncluded=true`
- `nativeAdmissionRankUnavailable=true`

页面显示“最低分换算位次”和“位次口径含政策加分”。这些位次不是院校投档或录取原表直接公布的原生最低位次。

## 证据

- 阳光高考《天津市 2025 年普通高考总成绩分数档（含政策加分）》
- 阳光高考保存的天津市教育招生考试院 PDF
- 中国教育在线完整 HTML 分数档表
- 天津 2025 普通高校招生工作规定

考试院原附件地址在当前会话直连时发生 TLS 错误，导入器因此记录为 `blocked-current-session-tls`；官方阳光高考页面/附件、完整 HTML 和政策文件共同闭合数值与加分口径。
