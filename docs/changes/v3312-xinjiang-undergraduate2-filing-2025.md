# v3.312 新疆 2025 普通类本科二批投档数据

## 目标

从新疆教育考试院官方图片表导入 2025 年普通类本科二批次院校投档情况，为新疆不同分数段补充可核验的院校投档边界、计划数、投档人数、最高分、最低分、平均分和同分排序项。

## 官方来源

- 公告页：<https://www.xjzk.gov.cn/c/2025-08-06/494702.shtml>
- 公告标题：新疆维吾尔自治区2025年普通高校招生普通类本科二批次投档情况
- 7 张官方原图及 OCR 中间证据共 249 个文件进入版本证据包；每张原图保存尺寸、字节数和 SHA-256。
- 图片表在内部 APFS 上经 macOS Vision OCR、固定列重建和人工逐行复核，不在外置盘上运行解析程序。

## 数据结果

- 新增 1076 条官方记录，历史类 472 条、物理类 604 条，涉及 616 所院校。
- 1060 条有实际投档结果，最低分范围 280-529；完整保留计划数、投档人数、最高分、最低分、平均分和同分排序项。
- 16 条官方显示投档人数为 0，只保存为 `admission-plan`；分数和位次字段保持为空，并排除出推荐分数池。
- 189 行经过人工修正复核；1076 个源行全部入库，重复 ID、跳过行、缺学校、缺计划/投档人数和跨科类院校名称不一致均为 0。

## 事实边界

- 这是省考试院院校投档线，不是最终专业录取分；不能据此断言某专业一定录取。
- 官方表没有最低位次。1076 条记录全部标记 `rankUnavailable=true`、`rankDerivedFromScore=false`，不通过最低分硬造位次。
- 官方表没有专业名称、选科要求和专业录取人数；网站继续要求考生回当年招生计划、院校章程和专业限制复核。
- 16 条未投档计划只用于显示计划事实，不参与分数可达性、学校推荐排序或录取概率计算。
- 投档线是历史边界，不等于下一年度录取概率；系统只输出证据分层、风险提示和理由，不输出伪精确概率。

## 验收

1. 严格导入器复核 7 张官方图片、1076 个候选行、1076 条记录、189 个人工修正行和 0 跳过行。
2. 1060 条投档线与 16 条未投档计划严格分层，最低分、最高分、计划数、投档人数和同分项字段完整。
3. 新疆 gzip 分片增至 11518 条，全国运行总量增至 868426 条；位次换算记录保持 116656 条，来源说明增至 5117 条。
4. 塔里木理工学院物理类样本保留最低分 368、最高分 426；广东培正学院未投档样本保持计划数 10、投档人数 0 且无分数。
5. 当前发布 58 项回归全部通过。
