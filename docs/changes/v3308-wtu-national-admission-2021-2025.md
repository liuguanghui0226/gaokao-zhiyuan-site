# v3.308 武汉纺织大学 2021-2025 全国录取数据

## 目标

从武汉纺织大学本科招生信息网官方历年录取分数查询导入 2021-2025 年分省、分科类、分专业数据，为全国候选池补充学校直接公布的录取人数、省线、最低分、最低位次、平均分、最高分、分差和选科备注。

## 官方来源

- 历年分数查询：<https://zsb.wtu.edu.cn/cx/fsx.jsp>
- 按 5 个年份和 31 个省级口径下载 155 个官方查询页，共解析 2222 条专业行和 481 条官网汇总行。
- 每个页面保存字节数和 SHA-256；原始 HTML 与清单进入版本证据包，不进入 GitHub Pages 运行分片。

## 事实边界

- 新增 2222 条记录，覆盖 2021-2025 年和全部 31 个省级口径。
- 1633 条普通记录为 `school-official-only`；589 条专项、艺体、中外合作、预科、定向等限定路径为 `special-path-only`。
- 1921 条最低位次来自学校官网表格，标记为 `rankEvidenceScope=school-recorded-min-score-rank` 和 `rankDerivedFromScore=false`；301 条源表无位次的记录保持 `rankUnavailable=true`。
- 官网“艺文/艺理”在检索层归一为艺术类，但证据对账保留原始科类；8 条原先可能被普通类计划名掩盖的表演行已按特殊路径隔离。
- 学校官网单校录取边界不能替代省考试院全量投档/录取表，也不能单独生成录取概率。
- 西藏 17 条记录均无最低位次，源表没有 A/B 类口径；不得解释为西藏省级正式闭合。

## 验收

1. 155 个官方页面、2222 条专业行、481 条汇总行、31 省和五年度计数完整，重复 ID、跳过行和汇总差异均为 0。
2. 普通与特殊路径严格隔离，原生最低分位次与分数换算位次严格区分。
3. 31 省 gzip 分片、核心索引和清单哈希一致，运行总量为 857225 条。
4. 江西 2025 年计算机类样本保留最低分 554、平均分 560.1、最高分 572、录取 7 人、官网最低位次 32276和“化学必选”；未选化学时不会进入普通推荐。
5. 当前发布 45 项回归全部通过。
