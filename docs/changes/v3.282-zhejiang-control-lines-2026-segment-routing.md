# v3.282 浙江 2026 各类别分数线与普通类分段路由

## 目标

把浙江省教育考试院公开的 2026 年普通高校招生各类别分数线结构化入库，并修正推荐器对浙江普通类第一段、第二段的语义：它们是考生分段，不是本科线、专科线。

- 同年一分一段表：<https://www.zjzs.net/art/2026/6/26/art_45_12452.html>

- 五张表共 57 条：普通分段 2、特殊类型 1、艺术文化 5、艺术综合分 22、体育综合分 2、单独考试招生 25。

## 推荐行为

- 494 分及以上显示“普通类第一段”，但继续按位次、当年计划和具体院校专业投档线排序。
- 266-493 分显示“普通类第二段”，同时保留剩余本科与高职专科候选，不按“只能读专科”处理。
- 265 分及以下显示“普通类第二段线以下”，只作征求志愿、高职单招、技能培养和复读再规划等路径调研；模型总分最高 42、可信度强制为 C。
- 其余 55 条特殊类型、艺术、体育和单独考试招生控制线全部保持 `special-path-only`，不进入普通类推荐。

## 事实边界

控制线只用于相应类别和分段资格判断，不是院校专业投档线、最终录取分或录取概率。艺术综合分、体育综合分、文化分和单独考试招生成绩必须按各自口径使用，不得混算。

## 验收

1. 官方 HTML 五张表稳定解析为 57 条，六类路由计数闭合。
2. 浙江分片由 110889 条增至 110946 条，全国记录由 846462 条增至 846519 条。
3. 265/266/493/494 分临界行为通过，第二段同时保留本科与高职候选，线以下封顶 42/C。
4. 全国 31 省、155 个分段场景、31 个真实推荐探针和浏览器分片回归通过。
