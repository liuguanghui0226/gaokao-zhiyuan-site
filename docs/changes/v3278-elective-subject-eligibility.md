# v3.278 再选科目资格筛选

## 目标

将新高考专业的再选科目要求纳入自动推荐，避免把明确要求化学、生物、政治或地理的专业推荐给不符合选科的考生。

## 规则

- 表单支持化学、生物、思想政治、地理的多选输入。
- 单科必选、任选一科和明确多科均须选考的要求可自动判断。
- 明确不符合的院校专业和计划记录不进入候选。
- 空白、未公开或多科表述不清的要求不强行排除，标记为“选科待核”。
- 可执行清单保留原始选科要求，供考生回官方招生目录核验。

## 验证

`node scripts/test-elective-requirement-v3278.mjs` 覆盖单科、任选、多科均须、政治同义词、表述含糊、录取记录与计划记录排除边界。真实江西分片中的湖南大学计算机科学与技术记录“首选物理，再选化学”在化学+生物画像下通过匹配。
