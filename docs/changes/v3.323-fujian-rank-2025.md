# v3.323 福建 2025 普通类公开分段位次对齐

## 变更

- 导入福建 2025 普通类历史、物理一分一段表，形成 932 条省级位次区间。
- 两份完整结构化表共 933 行，全部通过逐分连续、同分人数累计闭合和单调性校验。
- 用福建省教育考试院 8 张 2480 x 3507 原图交叉复核 930 个正人数分数档、2790 个数据单元。

## 边界

- 表中累计人数按含政策性加分的投档成绩统计，只用于福建 2025 同首选科目的普通类分数位次换算。
- 历史 672-750 分只保存 1-14 名，物理 689-750 分只保存 1-46 名，不生成合并档内伪精确名次。
- 历史 664 分为 0 人；结构化转载表保留占位，考试院原图省略，本轮不生成空位次区间。
- 官方原图公开至 215 分。1 条低于公开下限的普通记录不向下外推，149 条 `special-path-only` 记录不连接普通类位次。
- “最低分换算位次”是同年省级分数段区间，不冒充院校投档表直接公布的原生最低位次，也不提升原最低分来源等级。
