# v3.273 西藏 2026 计划路径安全分流

## 目标

核验西藏自治区教育考试院通过阳光高考发布的《西藏自治区 2026 年普通高等学校招生计划》已完整进入全国运行层，并将该全量计划中的限定资格路径从普通自动推荐中安全分流。

本轮不重复导入数据。运行分片已经有 7,300 条该来源的专业级计划记录；原公告首页总计划为 30,905 名，当前结构化来源行的计划数为 30,903 名，是 v3.270 三峡大学中外合作办学更正替换原计划后的记录口径。重复导入会造成计划计数和候选重复。

## 不改变的边界

- 这是招生计划，`planCount` 不等于投档数、录取数、录取最低分或录取概率。
- 普通、军警、专项、预科、艺体、部队生源和对口高职计划必须保存各自路径；不把特殊路径混入普通候选。
- 西藏仍没有公开可计算的一分一段、省级普通批全量投档/录取表、专业最低位次及高职专科投档表，不能声称西藏普通批录取概率已经闭合。

## 权威来源

- 阳光高考转载的西藏自治区教育考试院公告：<https://gaokao.chsi.com.cn/gkxx/zc/ss/202606/20260626/2293847302.html>
- 同页附件：《西藏自治区 2026 年普通高等学校招生计划》DOCX。

该附件首页明确总计划 30,905 名，其中普通生源 26,778 名、部队生源 250 名、对口高职 3,877 名；导入器必须以专业明细计划和上述总量交叉校验。

## 受影响的表面

| 表面 | 影响 |
|---|---|
| 信息 | 保留既有专业级 `admission-plan` 记录，新增运行时限定资格判定。 |
| 用户 | 普通西藏画像只获得可按普通路径填报的 2026 专业计划，限定路径需确认资格后另行核验。 |
| 运行 | 更新推荐器、测试、镜像中的前端资产与 HTTP 静态运行层。 |

## 验收

1. 原始页面和 DOCX 的标题、发布机构、表头和 30,905 名声明可复核；现有来源行、v3.270 更正和运行分片计数一致。
2. 提前、专项、预科、艺体、部队、对口和定向等限定资格计划不进入普通候选。
3. 普通计划不冒充录取分或概率，数据面板明确显示被隔离的限定路径数量。
4. 前端资产、镜像、HTTP 与逻辑测试的版本和边界一致。
