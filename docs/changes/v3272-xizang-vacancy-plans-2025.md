## 规格

### 目标

接入阳光高考转载且明确标注“来源：西藏自治区教育考试院”的 2025 年 12 次征集志愿公告及 23 个附件，并接入 2026 年普通高校招生录取工作进度计划。每个专业记录保存当轮剩余计划、科类、批次、学制、学费、限制条件和来源附件；推荐模型只把它用于发现历史补录机会和核验专业池，不把它当成录取最低分、最低位次或下一年一定存在的计划。

### 验收条件

- [x] AC1：固定解析 12 个 2025 公告页面、22 个 DOCX 和 1 个 XLS；初始地址和最终响应地址只允许 `gaokao.chsi.com.cn`、`t1.chei.com.cn` 至 `t4.chei.com.cn`。
- [x] AC2：每个原始文件保存字节数和 SHA-256；`--use-cache` 重放结果稳定，缓存篡改、错误主机和跨域重定向均被拒绝。
- [x] AC3：专业记录逐项保存 `planStage=征集志愿`、公告轮次、当轮剩余计划、学费、备注和 A/B 类资格线；所有记录均没有 `minScore`、`minRank`，不得进入录取可达性计算。
- [x] AC4：普通本科、普通专科与对口、部队生源、民族预科、艺体等特殊入口分开；`special-path-only` 不进入普通类计划候选。
- [x] AC5：同一院校专业跨轮再次出现时保存 `vacancyRepeatCount`，但年度 `admissionPlanCount` 不累加重复剩余席位。
- [x] AC6：基础段可看到匹配兴趣的普通专科征集机会，高分本科画像不混入专科征集；页面明确提示“历史剩余计划快照，不是录取概率”。
- [x] AC7：西藏推荐页显示计划最新年、普通录取数据最新年、位次换算最新年及 2026 当前录取阶段；缺一分一段时要求用户填写考试院正式位次。
- [x] AC8：31 省分片、APFS 主库、mac_2T 运行镜像、HTTP 回读和浏览器交互全部守恒，独立只读 Reviewer 无未关闭 P1/P2。

### 非目标与边界

- 不把征集公告中的“A/B 类考生不低于某分”写成院校或专业录取最低分。
- 不把多轮剩余计划相加为年度招生计划，也不推断下一年仍会征集。
- 不通过考生账号、验证码或非公开接口抓取西藏辅助系统。
- 不宣称西藏普通类正式数据闭合；仍缺公开可计算一分一段、省级全量投档/录取表和专业最低位次。
- 不改动其他 30 省已有录取、计划或位次记录。

### 假设与开放问题

- 高置信假设：阳光高考转载页的“来源：西藏自治区教育考试院”和原附件可作为省级官方发布镜像；导入器同时锁定页面、附件和哈希。
- 中置信假设：DOCX 中空学制/计划的行是院校汇总行，带专业代码的行是专业记录；通过表头、院校合计和公告总人数交叉验证。
- 非阻塞开放项：西藏考试院主站当前网络路径返回 502，辅助系统导航超时；本轮不改变 VPN/代理，也不绕过公开访问边界。

## 实施计划

### 技术路径与依赖顺序

1. 在 Node 导入器中声明 12 个公告、23 个附件、批次和普通/特殊入口分类，下载并锁定响应主机、字节数和 SHA-256。
2. 用 `unzip -p ... word/document.xml` 解析 DOCX 表格；用 LibreOffice 无头转换 XLS 为 CSV，再由 Node CSV 解析器读取。
3. 校验公告标题、科类、院校合计、专业合计、公告人数和字段契约，生成稳定 ID、跨轮重复次数及两个来源注。
4. 用 Node 多行流式刷新主库；同目录临时文件通过全部计数/字段校验并 `fsync` 后再原子替换，只增加记录数与 `vacancyPlanRecords`，不增加年度 `admissionPlanCount`，不改变西藏正式分数/位次闭合状态。
5. 扩展省分片字段白名单和推荐界面：征集机会专属文案、学历路径隔离、特殊入口隔离、数据新鲜度面板。
6. 重建分片，镜像到 mac_2T，完成 Node、HTTP、浏览器及独立审查。

### 回滚

- 模式：1，可逆文件替换。
- 机制：恢复 v3.271 APFS 硬链接快照，重建分片并重新镜像；不运行旧版回滚型 finalizer。
- 数据保护：主库替换前校验基线记录数、目标来源 ID 不存在和输出 JSON 完整性。

## 实施结果

- 官方供应链锁定 12 个公告页、22 个 DOCX、1 个 XLS 和 2026 日程页/图片，共 37 个原始文件哈希；缓存衍生 CSV 每次从固定 XLS 重建并校验。
- 导入 2187 条专业级征集记录，其中普通路径 2157 条、特殊路径 30 条、普通高职专科 926 条；6099 个当轮剩余计划快照只作历史补录信号，不计入年度招生计划。
- 1560 条记录属于 742 个跨轮重复组；23 条为数字媒体技术。全部 2187 条记录的 `minScore`、`minRank` 字段计数均为 0。
- 全国运行层保持 31 个省级分片、843963 条录取/计划记录和 116656 条位次换算；西藏分片为 28298 条记录，其中本批 2187 条完整守恒。
- 180 分画像不越过 202 分资格线；250 分但 A/B 类未知时显示“资格待核验”；650 分普通本科画像的高职征集候选为 0。
- 流式刷新器覆盖成功原子替换、计数失败、缺失审计、三组路径冲突和符号链接别名；所有失败都保留旧输出，临时文件残留为 0。
- 推荐说明统一规范句号/分号边界并为 v3.272 脚本加缓存版本参数，最终页面不再出现连续标点。
- 独立只读 Reviewer 的多轮检查已关闭缓存衍生物信任、学历路径泄漏、边境专项、资格线语义、重复键和刷新失败安全等全部 P1/P2，最终结论 PASS。
