# v3.315 河北 2025 一分一档证据

本目录记录河北 2025 年历史类、物理类一分一档表的来源边界、文件哈希、逐单元格 OCR 和完整表交叉核验结果。18 页官方 PDF、OCR 文本、转载索引与独立完整表打包为 GitHub Release 资产 `evidence-v3.315-hebei-rank-2025.tar.gz`，网页运行分片只发布结构化结果。

## 核验结论

- 河北省教育考试院官方 PDF 为 18 页普通类成绩统计表；中国教育在线转载索引标明来源为河北省教育考试院，并链接同一 PDF 文件。
- PDF 表格逐单元格 OCR 提取历史类 533 个连续分数档、物理类 554 个连续分数档。
- 1087 个累计位次单元与独立完整表逐项一致，无重复分数、无断档，累计人数单调，公布人数闭合。
- 最高分行是合并档：历史类 672 分及以上为 1-35 名，物理类 693 分及以上为 1-32 名。
- 一分一档表只能把分数换算为省级分数段位次；它不证明院校投档或录取原表直接公布了最低位次。

机器可读详情见 `evidence-manifest.json`。
