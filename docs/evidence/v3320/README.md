# v3.320 广西 2025 双加分口径一分一档表证据

本目录记录广西 2025 年普通高考历史类、物理类一分一档表的区内/区外加分口径、跨源核验和院校所在地判定边界。参与核验的 25 份原始文件打包为 GitHub Release 资产 `evidence-v3.320-guangxi-rank-2025.tar.gz`，网页运行分片只发布结构化结果。

## 核验结论

- 阳光高考两科页面明确标注来源为广西招生考试院，并保存区外院校全国性加分和区内院校最高一项加分的四组图片。
- 四份 XLSX 与两个 HTML 完整表逐行比较，1892 个公开分数档、7568 个单元差异为 0。
- 加上四个官方未逐分披露的最高分合并档，共保存 1896 条位次区间；区内、区外各 948 条。
- 教育部 2025 全国普通高校名单共 2919 所，其中广西 89 所，本科 41 所、高职专科 48 所。
- 8222 条广西 2025 普通整数最低分按目标院校所在地连接对应位次；110 条特殊路径不生成普通位次。
- 换算结果统一标为“最低分换算位次”，不冒充院校原表直接公布的录取最低位次。
- Playwright 在桌面和手机视口验证广西区外/区内分数与位次字段、自动估算和无横向溢出；结果与截图见本目录 `browser-qa.json`、`desktop-guangxi-rank.png` 和 `mobile-guangxi-rank.png`。

机器可读详情见 `evidence-manifest.json`。
