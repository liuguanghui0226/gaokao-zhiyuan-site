# v3.319 江苏 2025 第一阶段一分一段表证据

本目录记录江苏 2025 年普通高考第一阶段历史、物理等科目类逐分段统计表的来源边界、文件哈希、逐格 OCR 和全表交叉核验结果。原始证据文件打包为 GitHub Release 资产 `evidence-v3.319-jiangsu-rank-2025.tar.gz`，网页运行分片只发布结构化结果。

## 核验结论

- 江苏省教育考试院发布页及两张 1588x4488 原图为主证据；阳光高考两张图片与考试院原图逐字节一致。
- 大学生必备网保存的两张全尺寸 PNG 与考试院原图的全表 Tesseract TSV 逐字节一致。
- 第一阶段共保留历史类 177 档、物理类 221 档，398 行、1194 个分数/同分人数/累计人数单元跨源差异为 0。
- 15 处披露的 OCR 校正均由连续分数网格或累计人数算术闭合恢复；累计人数严格递增，每行人数闭合。
- 历史类 658 分及以上只保存 1-109 名，最低公布到 482 分；物理类 683 分及以上只保存 1-126 名，最低公布到 463 分。
- 第二阶段表只统计仍符合第二阶段志愿填报条件的考生，不是全体同科类考生累计位次，已排除且不与第一阶段拼接。
- 7060 条普通最低分记录连接到省级分数段位次；1496 条低于公布下限的普通记录和 123 条特殊路径记录不生成假位次。
- 换算结果统一标为“最低分换算位次”，不冒充院校原表直接公布的录取最低位次，也不用于艺术、体育综合分。

机器可读详情见 `evidence-manifest.json`。
