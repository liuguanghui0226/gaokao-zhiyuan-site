# v3.314 宁夏 2025 一分一段证据

本目录记录宁夏 2025 年历史组、物理组一分一段表的来源边界、文件哈希、双镜像一致性和解析检查点。原始网页、两组 PDF 镜像副本与文本层打包为 GitHub Release 资产 `evidence-v3.314-ningxia-rank-2025.tar.gz`，网页运行分片只发布结构化结果。

## 核验结论

- 阳光高考转载页注明来源为宁夏教育考试院，并列出两份 PDF 的准确文件名。
- 历史类和物理类 PDF 均从两个独立域名取得；每科两份文件逐字节一致。
- 历史类解析 467 个连续分数档，物理类解析 492 个连续分数档；无重复分数、无断档，累计人数单调。
- 最高分行是合并档：历史 616 分及以上为 1-54 名，物理 641 分及以上为 1-104 名。
- 这两张表只能把分数换算为省级分数段位次，不证明院校原表直接公布了录取最低位次。
