# v3.323 福建 2025 普通类一分一段表证据

本目录记录福建 2025 年普通高考历史、物理科目组一分一段表的官方来源、跨载体核验和运行边界。参与核验的 36 份原始文件已打包为 GitHub Release 资产 `evidence-v3.323-fujian-rank-2025.tar.gz`，网页运行分片只发布结构化结果。

## 核验结论

- 福建省教育考试院两科发布页共列出 8 张官方原图，页面标题、来源和发布日期完整。
- 中国教育在线两份完整表明确标注来源为福建省教育考试院；共解析 933 行，逐分连续、同分人数与累计人数逐行闭合。
- 8 张考试院原图实际刊出 930 个正人数分数档，按三列固定网格 OCR 形成 2790 个单元审计。结构化数值以算术闭合的完整表为基线，原图用于独立交叉复核。
- 历史 457 条、物理 475 条，共保存 932 条位次区间；两科公开到 215 分。
- 历史 664 分为 0 人占位，考试院原图未刊出，不生成空位次区间。
- 7591 条福建 2025 普通整数最低分连接同年位次；149 条特殊路径和 1 条低于公开下限的普通记录不生成普通位次。
- 换算结果统一标为“最低分换算位次”，不冒充院校原表直接公布的录取最低位次。
