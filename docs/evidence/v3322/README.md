# v3.322 湖北 2025 普通类一分一段表证据

本目录记录湖北 2025 年普通高考首选历史、首选物理一分一段表的官方来源、跨载体核验和运行边界。参与核验的 54 份原始文件已打包为 GitHub Release 资产 `evidence-v3.322-hubei-rank-2025.tar.gz`，网页运行分片只发布结构化结果。

## 核验结论

- 湖北省教育考试院发布页列出 12 张一分一段原图并标注来源为湖北省招办；湖北省教育厅附件页提供历史、物理两份官方 PDF。
- PDF 文本层共解析 1311 行、3933 个数据单元，累计人数逐行闭合，分数连续下降，累计位次连续。
- 12 张考试院原图按三列固定网格 OCR；累计人数精确命中 1289 行，分数与累计人数同时命中 1260 行。结构化数值以通过算术闭合的官方 PDF 为准，原图用于独立交叉复核。
- 历史 650 条、物理 663 条，共保存 1313 条位次区间；两科都覆盖到 0 分公开档。
- 7659 条湖北 2025 普通整数最低分连接同年位次；196 条特殊路径不生成普通位次。
- 档分包含政策性加分；最高分未逐分披露部分只保存官方累计范围。
- 换算结果统一标为“最低分换算位次”，不冒充院校原表直接公布的录取最低位次。

机器可读详情见 `evidence-manifest.json`；54 份文件的逐文件 SHA-256 也固化在 `data/admissions/official-hubei-rank-conversion-2025-v3322-import.json`。
