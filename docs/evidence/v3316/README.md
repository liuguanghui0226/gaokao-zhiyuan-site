# v3.316 重庆 2025 一分一段证据

本目录记录重庆 2025 年历史类、物理类一分一段表的来源边界、文件哈希和全表交叉核验结果。六份来源网页和机器清单打包为 GitHub Release 资产 `evidence-v3.316-chongqing-rank-2025.tar.gz`，网页运行分片只发布结构化结果。

## 核验结论

- 中国教育在线索引标注来源为重庆市教育考试院，并保留历史类、物理类考试院原始表链接；第二个转载索引保留相同链接。
- 当前会话直接访问考试院表时 TLS 不可用，因此来源等级明确写为“考试院归属内容镜像、双索引保留原始链接、独立完整表复核”，不声称官网直连成功。
- 中国教育在线完整表与大学生必备网独立完整表的历史类 473 行、物理类 502 行全部一致；975 行共 2925 个分数、同分人数、累计人数单元零差异。
- 两科分数连续、累计人数单调、同分人数闭合。历史类 652-180 分累计到 73373 人，物理类 681-180 分累计到 139478 人。
- 最高分行是合并档：历史类 652 分及以上为 1-66 名，物理类 681 分及以上为 1-159 名。
- 一分一段表只能把分数换算为省级分数段位次；它不证明院校投档或录取原表直接公布了最低位次。

机器可读详情见 `evidence-manifest.json`。
