# v3.318 湖南 2025 一分一段表证据

本目录记录湖南 2025 年历史、物理科目组合一分一段表的来源边界、文件哈希和全表交叉核验结果。九份原始证据文件打包为 GitHub Release 资产 `evidence-v3.318-hunan-rank-2025.tar.gz`，网页运行分片只发布结构化结果。

## 核验结论

- 中国教育在线两份完整逐分表均标注来源为“湖南省教育厅”；阳光高考两科页面标注来源为“湖南考试招生”并保存原图。
- 湖南省教育厅发布会明确说明档分含政策性加分，并确认 600 分以上历史类 2369 人、物理类 15860 人。
- 来源标注完整表共 1152 行，逐行满足“上一档累计人数 + 本档人数 = 本档累计人数”。
- 独立镜像覆盖 1139 行、3417 个分数、同分人数和累计人数单元，与来源标注表差异为 0。
- 历史类 9 个、物理类 2 个零人数分数保留为空档；历史 658-750 分只保存 1-55 名，物理 690-750 分只保存 1-53 名。
- 0-99 分是合并档，不用于普通录取最低分换算。网站最低换算到 100 分。
- 换算结果统一标为“最低分换算位次”，不冒充院校投档或录取原表直接公布的原生最低位次，也不用于艺术、体育综合分。
