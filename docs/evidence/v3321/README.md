# v3.321 山西 2025 普通类一分一段表证据

本目录记录山西 2025 年普通高考历史类、物理类本科控制线及以上一分一段表的来源归属、跨源核验和运行边界。参与核验的 18 份原始文件打包为 GitHub Release 资产 `evidence-v3.321-shanxi-rank-2025.tar.gz`，网页运行分片只发布结构化结果。

## 核验结论

- 阳光高考索引和两科页面明确标注来源为山西省招生考试管理中心；晋城市政府转载页同时保存两科表并标注发布者。
- 中国教育在线与大学生必备网两套完整结构化表共比较 515 行、1545 个分数、同分人数、累计人数单元，差异为 0。
- 官方归属原图 OCR 精确命中 509 个分数单元和 514 个累计人数单元；7 处已知识别例外均由连续分数网格、累计闭合或独立表复核。
- 历史 230 条、物理 287 条，共保存 517 条位次区间；公开下限分别为历史 443 分、物理 419 分。
- 6587 条山西 2025 普通整数最低分连接同年位次；75 条特殊路径不生成普通位次，低于公开下限的 2100 条普通记录不外推。
- 8 月未录取考生成绩分段表作为排除证据保存，不与 6 月全体普通类表混合。
- 换算结果统一标为“最低分换算位次”，不冒充院校原表直接公布的录取最低位次。
