# v3.325 海南 2025 普通类一分一段表证据

本目录记录海南 2025 年普通高考普通类综合投档分表的官方来源、双分辨率识别和运行边界。86 份源文件与固定网格 OCR 已打包为 GitHub Release 资产 `evidence-v3.325-hainan-rank-2025.tar.gz`；网页运行分片只发布结构化结果。

## 核验结论

- 海南省考试局页面直接发布 21 页 4761x6733 原图，标题明确投档成绩为高考成绩综合分与照顾加分之和。
- 政府账号转载页提供同一 21 页表的 1080x1527 独立缩放图，并标注来源为海南省考试局。
- 800 分及以上为 105 人；799-246 分共 554 个逐分档。
- 两套 OCR 的 549 个累计位次直接一致；7 个异常格经镜像值或相邻人数闭合纠正。
- 所有最终累计位次严格递增，相邻累计差全部为正人数；246 分累计 67408 人。
- 555 条位次区间均使用综合类、含照顾加分口径。
- 4241 条海南 2025 综合普通类整数最低分连接同年位次；35 条特殊路径不连接普通位次。
- 换算结果统一标为“最低分换算位次”，不冒充院校原表直接公布的录取最低位次。

机器可读详情见 `evidence-manifest.json`；逐图 SHA-256 固化在 `data/admissions/official-hainan-rank-conversion-2025-v3325-import.json`。
