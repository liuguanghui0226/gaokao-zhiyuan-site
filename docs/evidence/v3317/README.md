# v3.317 辽宁 2025 成绩统计表证据

本目录记录辽宁 2025 年历史类、物理类成绩统计表的来源边界、文件哈希和全表交叉核验结果。十份证据文件打包为 GitHub Release 资产 `evidence-v3.317-liaoning-rank-2025.tar.gz`，网页运行分片只发布结构化结果。

## 核验结论

- 辽宁省教育厅转载索引明确标注文章来源为“辽宁招生考试之窗”，并链接考试院两科原始 PDF。
- 当前会话直接访问考试院附件时 TLS 不可用；阳光高考保存的两份辽宁招考办 PDF 镜像可正常读取，来源等级明确写为“官方内容镜像”，不声称官网直连成功。
- PDF 文本层与中国教育在线两科完整表的历史类 517 行、物理类 556 行全部一致；1073 行共 3219 个分数、同分人数、累计人数单元零差异。
- 历史类 667、164、162 分和物理类 703、153 分均为零人数缺口，保留为缺口而不生成虚假位次行。
- 历史类 669 分及以上为 1-10 名，物理类 707 分及以上为 1-11 名；两科最低展示到 150 分。
- 表中人数包含同科类普通、艺术和体育考生。只用于同年同科类文化分到省级位次区间的换算，不用于艺术、体育综合分。
- 成绩统计表不证明院校投档或录取原表直接公布了最低位次，网站统一标为“最低分换算位次”。

机器可读详情见 `evidence-manifest.json`。
