# v3.324 黑龙江 2025 普通类一分一段表证据

本目录记录黑龙江 2025 年普通高考历史、物理类文化课一分一段表的官方来源、跨载体核验和运行边界。7 份稳定原始证据已打包为 GitHub Release 资产 `evidence-v3.324-heilongjiang-rank-2025.tar.gz`；网页运行分片只发布结构化结果。

## 核验结论

- 黑龙江省政府发布页列出黑龙江省招生考试院历史、物理两份官方 XLS，并明确统计不含照顾政策分。
- XLS 共 1093 个表格行；1091 个正人数分数档生成位次记录。
- 两份完整表共 1094 行，1091 个正人数档与 XLS 比较；3273 个分数、人数、累计人数单元差异为 0。
- 历史类 528 条、物理类 563 条，共保存 1091 条位次区间；两科公开到 130 分。
- 历史 654、144 分和物理 136、134 分为零人数空档，不生成位次记录。
- 历史 659-750 分只保存 1-19 名，物理 694-750 分只保存 1-34 名。
- 7098 条黑龙江 2025 普通整数最低分连接同年位次；84 条特殊路径不连接普通位次。
- 换算结果统一标为“最低分换算位次”，不冒充院校原表直接公布的录取最低位次。

机器可读详情见 `evidence-manifest.json`；逐文件 SHA-256 固化在 `data/admissions/official-heilongjiang-rank-conversion-2025-v3324-import.json`。
